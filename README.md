# Prashant Kumar Misra Portfolio

Static personal portfolio website generated from the resume at:

`D:\Downloads\Prashant_Misra_AI_Resume_11_05.docx`

## Open locally

Double-click `index.html`, or run a local server from this folder:

```powershell
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173
```

## Files

- `index.html` - page content and sections
- `styles.css` - responsive visual design
- `script.js` - animated AI systems canvas
- `resume.pdf` - downloadable resume
- `favicon.svg` - browser tab icon
- `preview.svg` - social sharing preview image
- `demos/` - local interactive demo pages for RAG, incident response, and forecasting

## Before publishing

- Replace the Formspree placeholder in `index.html` with your real form endpoint.
- Replace local demo shells with deployed backend URLs when your AI services are hosted.
